# task-1-adv
مشروع تعليمي يحاكي نظام إدارة مكتبة إلكترونية باستخدام مفاهيم البرمجة غرضية التوجه (OOP)
المشروع يتيح للمستخدم : 
-البحث عن الكتب باستخدام الاسم او المؤلف
-فلترة الكتب باستخدام قائمة منسدلة
-تغيير حالة توفر الكتاب
-حذف كتاب معين من المكتبة
-اضافة كتاب جديد

--تم تطبيق في هذه المشروع :
-الوراثة
-التغليف لحماية البيانات الداخلية
-تعددية الاشكال 
-التجريد
